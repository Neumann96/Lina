# Lina — Figma Import Kit

## Импорт редактируемого макета

1. Установите в Figma плагин **html.to.design**.
2. Создайте пустой Figma-файл и запустите плагин.
3. Выберите импорт веб-страницы по URL.
4. Вставьте `http://5.129.195.206/figma-import`.
5. Сначала импортируйте ширину `1440 px`, затем `390 px`.

Плагин преобразует DOM страницы в редактируемые Figma-слои. PNG из папки `previews` используйте как визуальный эталон.

## Состав комплекта

- `design-tokens.json` — цвета, радиусы, тени, отступы и типографика;
- `screen-manifest.json` — размеры и URL экранов;
- `assets/*.svg` — отдельные векторные иконки и знак Lina;
- `previews/*.png` — контрольные desktop/mobile-рендеры.

## Как вернуть дизайн разработчику

Передайте публичную Figma-ссылку с правом просмотра и экспортируйте готовые фреймы в PNG масштаба 1×. Новые логотипы и иконки приложите отдельными SVG. Имена основных фреймов: `Home / Desktop` и `Home / Mobile`.
